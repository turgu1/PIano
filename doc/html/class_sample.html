Detailed Description

Class Sample represents the static portion of a single note sampled at a specific gain (volume) level. (Class Voice represents the dynamic portion of it while it is played by the user.) When a sample is created, a first portion of the PCM data is gathered in memory and will be served while PIano setup itself to retrieve the rest of it when required through multithreading.
