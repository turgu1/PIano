Detailed Description

template<class T>
class NewHandlerSupport< T >

The NewHandlerSupport class supplies mixin to simplify memory allocation process when an out of memory event occurs. It supplies a modified new operator that manage the C++ new_handler feature. The class that inherit the mixin must supply it's own outOfMemory static method that will do the appropriate actions when a memory allocation exception occurs.
